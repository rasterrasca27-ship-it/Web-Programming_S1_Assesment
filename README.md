# Raster_S1_Exercises_Assessment

This repository contains the completed outputs for the **Set Exercises assessment** for the Web Development module.

## Exercise Folders

**01-semantic-markup**
A semantic HTML webpage demonstrating the correct use of structural HTML elements such as header, navigation, sections, and footer.

**02-css-styling**
A styled single-page article that applies external CSS for layout, colours, typography, and visual presentation.

**03-page-layout**
A media-rich gallery page that demonstrates page layout techniques using images and video.

**04-web-accessibility**
An accessibility-focused webpage redesign that includes semantic structure, descriptive alt text, and improved keyboard navigation.

## Notes

* Each exercise is self-contained and includes its own **index.html** file.
* CSS styling is separated into **style.css** where appropriate.
* Local media assets (images or videos) are stored inside the relevant exercise folders so the pages work independently.
* The repository is organised so that each exercise is clearly structured and easy to locate.
